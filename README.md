### Description
**It's just a study project**. The code that's written in this project doesn't pretend to be perfect.
It has some visual bugs, but it works, and you can actually play vs your friend or bot.

Background is very immesrive. Foreground and background mountains with dynamic shadow,
wind currents and kind of changing the time of day (even if it's not exactly in the timing)

Background state storaged in LocalStorage and you can restore your page (CTRL + R or F5)
with logo animation.
### Bots
It's just minimax algorithm implementation and a pinch of randomness that depends on
difficulty of the bot that you choose.
